# Vearch

## 🔎Vearch = Video + Search
- 동영상 내 키워드 검색을 가능하게 하는 크롬 확장 프로그램입니다. 
- 웹에서 재생되는 동영상의 오디오를 추출하여 STT 변환 후, 키워드의 등장 시간과 횟수를 한 눈에 볼 수 있습니다.
